# Book Tower (북타워) 요구사항 명세서

## 1. 프로젝트 개요

- **프로젝트명**: Book Tower (Pomodoro)
- **앱의 목적**: 사용자가 공부/집중 시간을 측정하면서, 물리 엔진 기반의 탑 쌓기(책 쌓기) 시각 효과를 통해 성취감을 얻고 몰입도를 높이는 뽀모도로(Pomodoro) 애플리케이션.
- **주요 타겟**: 집중력 향상이 필요한 학생, 수험생 및 직장인

---

## 2. 기술 스택 (Tech Stack)

- **프론트엔드 (Frontend)**:
  - HTML5, CSS3 (Vanilla CSS)
  - JavaScript (Vanilla JS)
- **물리 엔진 (Physics Engine)**:
  - `Matter.js` (2D 물리 엔진, 책 낙하 및 탑 붕괴 시뮬레이션)
- **폰트 (Typography)**:
  - Pretendard (가독성 높은 모던 폰트)
- **데이터 저장 (Storage)**:
  - `localStorage` (사용자 설정, 공부 시간 통계, 도감 수집 정보 로컬 영구 저장)
- **모바일/플랫폼 (Platform)**:
  - Android APK 빌드 타겟 (Cordova / Capacitor 연동을 위한 모바일 최적화 및 안드로이드 센서 제어 고려)

---

## 3. 기능 요구사항 (Functional Requirements)

|      번호      | 기능 내용                                                                              |
| :-------------: | :------------------------------------------------------------------------------------- |
| **FR-01** | 사용자는 일반 모드와 뽀모도로 모드 중 하나를 선택하여 타이머를 시작할 수 있다          |
| **FR-02** | 사용자는 타이머 진행 중 위에서 떨어지는 책으로 탑이 쌓이는 시각 효과를 볼 수 있다      |
| **FR-03** | 사용자는 기기의 자이로 센서를 통해 움직임이 감지될 경우 부정행위 경고(방해물)를 받는다 |
| **FR-04** | 사용자는 대시보드에서 오늘 방어 시간, 최고 탑 높이 등 누적 통계를 확인할 수 있다       |
| **FR-05** | 사용자는 월간 기록 달력에서 일별 공부 시간을 히트맵 형태로 확인할 수 있다              |
| **FR-06** | 사용자는 과목을 추가하고 색상을 커스텀하여 과목별 공부 시간을 측정하고 관리할 수 있다  |
| **FR-07** | 사용자는 집중 시간에 따라 무작위로 떨어지는 가챠 아이템을 획득할 수 있다               |
| **FR-08** | 사용자는 도감 뷰에서 자신이 획득한 아이템 목록을 확인할 수 있다                        |
| **FR-09** | 사용자는 실시간 랭킹에서 자신의 점수와 다른 사용자의 순위를 확인할 수 있다             |
| **FR-10** | 사용자는 랭킹 및 UI에 노출될 자신의 닉네임을 설정하고 변경할 수 있다                   |
| **FR-11** | 사용자는 라이트 모드, 다크 모드, 시스템 기본 테마 중 하나를 선택하여 적용할 수 있다    |
| **FR-12** | 사용자는 알림 및 물리 충돌 시 기기의 진동 발생 여부를 설정할 수 있다                   |
| **FR-13** | 사용자는 설정 메뉴에서 기기에 저장된 모든 데이터를 초기화할 수 있다                    |
| **FR-14** | 사용자는 앱 최초 실행 시 주요 기능과 사용법을 안내하는 튜토리얼을 볼 수 있다           |

## 4. UI/UX 디자인 요구사항 (UI/UX Guidelines)

- **디자인 철학 (Toss Style)**:
  - 깔끔하고 직관적인 모바일 스타일의 인터페이스.
  - 여백의 미, 둥근 모서리, 그림자 활용.
  - **메인 컬러**: Toss Blue, 배경: Light Gray, 텍스트: Dark Gray.
  - **다크 모드 지원**: 배경, 카드 등 눈부심 방지를 위한 어두운 테마 완벽 호환.
- **인터랙션 및 애니메이션**:
  - 버튼 클릭 시 눌림 효과.
  - 백그라운드 파티클 및 그라데이션 애니메이션 (우주 공간 느낌의 스테이지 변화 적용).
  - 모달 등장/퇴장 시 뒷배경 블러 효과 및 부드러운 트랜지션 적용.
- **네비게이션 (GNB)**:
  - 화면 하단 고정 Bottom Navigation Bar 제공 (홈, 아이템, 랭킹, 설정 탭).
  - 현재 활성화된 탭 아이콘 강조.

---

## 5. 예외 처리 및 엣지 케이스 (Edge Cases)

- **백그라운드 실행 시**: 앱이 백그라운드로 전환되어도 타이머가 정상적으로 카운트되도록 타임스탬프 차이 계산 로직 적용.
- **휴식 시간 오버랩 방지**: 뽀모도로 모드에서 휴식 시간이 중복으로 실행되지 않도록 상태 검증 철저.
- **자이로 센서 권한**: 기기에 따라 자이로 센서 권한이 명시적으로 필요한 경우 권한 요청 플로우 표시.
- **데이터 마이그레이션**: 이전 버전의 데이터를 불러올 때 하위 호환성을 유지하여 데이터 포맷을 최신 구조로 안정적으로 마이그레이션 처리.
